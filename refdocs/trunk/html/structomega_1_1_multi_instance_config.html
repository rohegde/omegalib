int omega::MultiInstanceConfig::id

application instance Id.

On systems running multiple omegalib instances concurrently, each application should have a different index set. The index is used, among other things, to apply offsets to communication ports, to avoid port conflicts. This number is set automatically during initialization by the omain() call. It is derived by command line parameters passed with the '-I' switch.
