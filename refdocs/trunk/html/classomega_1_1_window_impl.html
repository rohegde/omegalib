Detailed Description

A Window represents an on-screen or off-screen drawable. A drawable is a 2D rendering surface, typically attached to an OpenGL context. A Window is a child of a Pipe. The task methods for all windows of a pipe are executed in the same pipe thread. The default window initialization methods initialize all windows of the same pipe with a shared context, so that OpenGL objects can be reused between them for optimal GPU memory usage.
