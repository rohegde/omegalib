This function will be called on the mainloop() iteration after *d_endpointAllocator is called, which lets subclasses do initialization.

(They can't do so during allocateEndpoint because it's called during the Connection constructor when their constructors haven't executed yet.)

This value is managed by the Endpoints, but we need exactly one copy per Connection, so it's on the Connection.
