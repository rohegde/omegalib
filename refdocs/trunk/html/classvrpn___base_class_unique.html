Detailed Description

INTERNAL class to hold members that there should only be one copy of even when a class inherits from multiple vrpn_BaseClasses because it inherits from multiple user-level classes.

Note that not everything in vrpnBaseClass should be here, because (for example) the registration of types should be done for each parent class.
