Detailed Description

Stores omegalib display configuration data.

Member Enumeration Documentation

enum omega::DisplayConfig::ConfigType
Enumerator:
ConfigPlanar 

Planar configuration: the display tiles are on the same plane.

The user won't have to Specify position and rotation for each tile.

ConfigCustom 

Custom configuration.

The user will have to manually specify position and rotation for each tile, or they will be provided by a custom DisplayConfigBuilder.
